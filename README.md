# 🛒 EassYBuY – Your Premium Shopping Destination

EassYBuY is a full-stack e-commerce web application where users can browse products, add them to cart, and place orders with a smooth and modern shopping experience.

---

## 🚀 Live Demo

🌍 Frontend: https://ecommerce-wine-theta-16.vercel.app/
⚙️ Backend: https://eassybuy-backend.onrender.com

---

## 📌 Features

### 🔐 Authentication

* User Registration & Login
* Secure authentication using JWT
* Password-based login system

### 🛍️ Product Management

* Browse featured products
* View product details
* Dynamic product listing

### 🛒 Cart System

* Add items to cart
* Increase / decrease quantity
* Remove items from cart

### 💳 Checkout & Payment

* Enter shipping address
* Multiple payment options:

  * UPI
  * Credit/Debit Card
  * Net Banking
  * Cash on Delivery

### 📦 Orders

* Place orders successfully
* View order history
* Track ordered items

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```
git clone https://github.com/your-username/ECOMMERCE.git
cd ECOMMERCE
```

---

### 2️⃣ Backend Setup

```
cd backend
npm install
```

Create `.env` file:

```
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
```

Run server:

```
npm start
```

---

### 3️⃣ Frontend Setup

* Open frontend folder
* Run using Live Server (VS Code)
